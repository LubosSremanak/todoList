## About project
The project uses the structure described in my article [Angular structure](https://lubossremanak.site/blog/article/0)

## Development

## Docker 
Run command
```
docker-compose up --build
```
## Local
Clone repo
```
git clone
```
Install dependencies
```
npm install
```
Run NG dev server
```
ng serve
```
